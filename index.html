Mars-Rover Project

Instructions

  1. Open Console
  2. Type in boundary to see rover, marked as 'R'
  3. Use commandLine(rover, 'string'), using a string to control rover.
    • For rover, type in rover1 to move R1
    • Type in rover2 to move R2
    • Note that both rovers are facing North by default
    • Type f to move forward
    • Type b to move backward
    • Type r to turn right
    • Type l to turn left
      • Example: commandLine('rfrffrbblb') is a valid string of commands
  4. You will be able to see the rovers move on the grid below!
