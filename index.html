Mars-Rover Project

Instructions

  1. Open Console
  2. Type in boundary to see rover, marked as 'R'
  3. Use commandLine('string'), using a string to control rover.
    • Type f to move forward
    • Type b to move backward
    • Type r to turn right
    • Type l to turn left
      • Example: commandLine('rfrffrbblb') is a valid string of commands
  4. Once you move, you can type boundary again to see where the rover is
